📐 Rules

?
O que sao Rules
Rules sao regras de validacao e formatacao que restringem o comportamento do Claude durante a execucao. Escritas como texto/codigo, sao injetadas no system prompt como constraints. Podem ser globais (aplicadas a tudo) ou vinculadas a Skills/Plugins especificos. Suportam importacao de URLs (ex: GitHub raw) e sincronizacao automatica.

🪝 Hooks

?
O que sao Hooks
Claude Code Hooks permitem executar codigo customizado em pontos especificos do ciclo de vida de um workflow. Sao automacoes event-driven: quando um evento acontece (ex: ferramenta chamada, sessao iniciada, tarefa concluida), o hook e disparado e executa uma acao — seja um comando shell, um prompt de LLM ou um subagente. Suportam matchers regex para filtrar por ferramenta/contexto, timeouts e execucao assincrona.
Event-Driven Automacao Claude Code Lifecycle

Handler Types — Comparacao
Tres formas de executar um hook
💻
command
Executa um comando shell no sistema. Recebe variaveis de ambiente com contexto da tool ($TOOL_NAME, $TOOL_INPUT). Retorna exit code.
exit 0 → continua
exit 2 → bloqueia
💬
prompt
Envia um prompt para o LLM processar. O resultado pode influenciar a decisao (bloquear/permitir). Ideal para analise contextual.
LLM analisa e decide
Resultado retorna como texto
🤖
agent
Cria um subagente Claude completo para executar a tarefa. Mais poderoso, mais custoso. O agente tem acesso a ferramentas.
Subagente autonomo
Pode usar tools e decidir

⚖ Rules vs Hooks

Aspecto Rules Hooks
Proposito Restringir comportamento via constraints no prompt Automatizar acoes em pontos do ciclo de vida
Mecanismo Texto injetado no system prompt Codigo executado em resposta a eventos
Quando atua Durante toda a execucao (prompt-level) Em momentos especificos (event-driven)
Vinculacao WorkflowStepRule (join table) WorkflowStepHook (join table)
Pode bloquear Indiretamente (via instrucoes) Diretamente (exit 2 = block)
Importacao URL / GitHub sync Templates pre-configurados
Escopo Global, Skill ou Plugin Global ou por Projeto
Endpoints API 8 rotas 12 rotas (incl. templates, events, preview)
MCP Tools 7 tools 10 tools
